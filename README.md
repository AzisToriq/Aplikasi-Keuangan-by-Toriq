Aplikasi ini dirancang untuk membantu pengguna mengelola keuangan pribadi atau bisnis kecil mereka dengan mudah dan efisien. Lacak pemasukan, pengeluaran, dan dapatkan laporan keuangan yang jelas untuk mengambil keputusan yang lebih baik.

## ğŸŒŸ Fitur Utama

Aplikasi ini hadir dengan serangkaian fitur untuk manajemen keuangan yang komprehensif:

### 1\. **Manajemen Transaksi (Pemasukan & Pengeluaran)**

  * **Pencatatan Cepat:** Catat setiap transaksi (Pemasukan/Pengeluaran) dengan cepat.
  * **Kategorisasi:** Setiap transaksi dapat dikelompokkan ke dalam kategori tertentu (misalnya, Makanan, Transportasi, Gaji, Investasi).
  * **Tanggal & Deskripsi:** Mencatat tanggal transaksi, jumlah, dan deskripsi/catatan opsional.
  * **Fitur CRUD:** Memungkinkan pengguna untuk **C**reate (Buat), **R**ead (Lihat), **U**pdate (Ubah), dan **D**elete (Hapus) transaksi.

### 2\. **Laporan Keuangan & Visualisasi**

  * **Ringkasan Periode:** Lihat ringkasan pemasukan dan pengeluaran harian, mingguan, bulanan, atau tahunan.
  * **Visualisasi Data:** Grafik atau diagram lingkaran untuk memvisualisasikan distribusi pengeluaran per kategori.
  * **Filter Fleksibel:** Laporan dapat difilter berdasarkan rentang waktu, kategori, atau jenis transaksi.

### 3\. **Manajemen Kategori**

  * Pengguna dapat menambah, mengedit, atau menghapus kategori pemasukan dan pengeluaran sesuai kebutuhan.
  * Kategori kustom memungkinkan penyesuaian yang lebih baik terhadap pola keuangan individu.

### 4\. **Dasbor (Dashboard)**

  * Menampilkan saldo total saat ini.
  * Menunjukkan ringkasan cepat (misalnya, total pengeluaran bulan ini vs. bulan lalu).
  * Daftar transaksi terbaru untuk akses cepat.

## ğŸ› ï¸ Panduan Penggunaan Proyek (Instalasi & Run)

Ikuti langkah-langkah berikut untuk menjalankan proyek ini di lingkungan pengembangan lokal Anda.

### Prasyarat

  * **Android Studio:** Versi terbaru direkomendasikan.
  * **JDK:** Java Development Kit (versi 11 atau lebih tinggi).
  * **Perangkat/Emulator Android:** Untuk menjalankan aplikasi.

### Langkah-langkah Instalasi

1.  **Clone Repositori:**
    Gunakan Git untuk mengkloning repositori ke mesin lokal Anda:

    ```bash
    git clone https://github.com/AzisToriq/Aplikasi-Keuangan-by-Toriq.git
    ```

2.  **Buka di Android Studio:**
    Buka folder proyek (`Aplikasi-Keuangan-by-Toriq`) menggunakan Android Studio.

3.  **Sinkronisasi Gradle:**
    Tunggu hingga Android Studio selesai mengunduh semua dependensi dan melakukan sinkronisasi Gradle. Anda mungkin perlu menekan tombol **"Sync Now"** jika diminta.

4.  **Jalankan Aplikasi:**

      * Pilih perangkat virtual (Emulator) atau perangkat fisik (pastikan mode Debugging USB aktif).
      * Klik tombol **"Run"** (ikon segitiga hijau) di toolbar Android Studio.

## ğŸ“ Cara Menggunakan Aplikasi

Setelah aplikasi terinstal, berikut adalah panduan dasar navigasi dan penggunaan:

### 1\. Dasbor Awal

  * Saat pertama dibuka, Anda akan melihat saldo total (mungkin **Rp 0** jika data kosong).
  * Anda akan melihat ringkasan singkat pemasukan dan pengeluaran periode saat ini.

### 2\. Mencatat Transaksi Baru

1.  Cari tombol **"+"** (atau tombol sejenis) yang biasanya terletak di pojok bawah.
2.  Pilih **Jenis Transaksi**: **Pemasukan** atau **Pengeluaran**.
3.  **Masukkan Jumlah**: Ketik nominal transaksi.
4.  **Pilih Kategori**: Pilih dari daftar kategori yang tersedia (atau buat kategori baru).
5.  **Tanggal**: Atur tanggal transaksi (default adalah hari ini).
6.  **Deskripsi (Opsional)**: Tambahkan catatan detail.
7.  Tekan tombol **"Simpan"** atau **"Tambahkan"**. Saldo total akan diperbarui secara otomatis.

### 3\. Melihat Laporan

1.  Navigasi ke tab **"Laporan"** atau **"Statistik"**.
2.  Pilih rentang waktu yang diinginkan (Harian, Bulanan, atau Kustom).
3.  Lihat grafik yang menunjukkan distribusi pengeluaran.
4.  Gunakan filter untuk menganalisis data berdasarkan kategori tertentu.
